# Analisis_de_Ventas_de_Videojuegos_y_Tendencias_por_Plataforma_-2011-2016-
Proyecto de análisis de ventas de videojuegos entre 2011 y 2016, evaluando el rendimiento por plataforma, región, género y clasificación. Incluye análisis estadístico para identificar patrones, tendencias y relaciones entre calificaciones y ventas.
